# Формулы ВКР FitApp

Этот файл содержит все математические формулы, используемые в пояснительной записке.
Для каждой формулы приведены: LaTeX, формат UnicodeMath (Word 2016+) и место вставки.

---

## Ф.1 — Формула Эпли (оценка 1ПМ)

**Раздел:** 4.1 «Постановка задачи»  
**Описание:** Классическая аппроксимация одноповторного максимума по поднятому весу W и числу повторений N.

**LaTeX:**
```latex
\hat{1RM} = W \cdot \left(1 + \frac{N}{30}\right)
```

**UnicodeMath (вставка через Alt+= в Word):**
```
1RM̂ = W·(1+N/30)
```

**Как вставить в Word:**
1. Поставьте курсор в нужное место.
2. Нажмите `Alt` + `=` → откроется поле формулы.
3. Введите: `1RM̂ = W·(1+N/30)` или используйте кнопку «Линейный» и наберите в Unicode-формате.

## Ф.2 — Целевая переменная (1ПМ через 28 дней)

**Раздел:** 4.2 «Набор данных»  
**Описание:** Определение целевой переменной для обучения — 1ПМ спустя 28 календарных дней.

**LaTeX:**
```latex
y_i = \hat{1RM}_{i+28d}
```

**UnicodeMath:**
```
y_i = 1RM̂_{i+28d}
```

---

## Ф.3 — MAE (средняя абсолютная ошибка)

**Раздел:** 4.4 «Результаты обучения»  
**Описание:** Основная метрика качества регрессионной модели.

**LaTeX:**
```latex
MAE = \frac{1}{n} \sum_{i=1}^{n} \left| y_i - \hat{y}_i \right|
```

**UnicodeMath:**
```
MAE = 1/n ∑_(i=1)^n |y_i - ŷ_i|
```

---

## Ф.4 — MAPE (средняя абсолютная процентная ошибка)

**Раздел:** 4.4 «Результаты обучения»  
**Описание:** Относительная метрика, не зависящая от абсолютных значений веса.

**LaTeX:**
```latex
MAPE = \frac{100\%}{n} \sum_{i=1}^{n} \left| \frac{y_i - \hat{y}_i}{y_i} \right|
```

**UnicodeMath:**
```
MAPE = 100%/n ∑_(i=1)^n |( y_i - ŷ_i)/y_i|
```

---

## Ф.5 — RMSE (среднеквадратическое отклонение)

**Раздел:** 4.4 «Результаты обучения»  
**Описание:** Вспомогательная метрика, штрафующая за крупные выбросы.

**LaTeX:**
```latex
RMSE = \sqrt{\frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2}
```

**UnicodeMath:**
```
RMSE = √(1/n ∑_(i=1)^n (y_i - ŷ_i)²)
```

---

## Ф.6 — Коэффициент детерминации R²

**Раздел:** 4.4 «Результаты обучения»  
**Описание:** Доля объяснённой дисперсии.

**LaTeX:**
```latex
R^2 = 1 - \frac{\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{n}(y_i - \bar{y})^2}
```

**UnicodeMath:**
```
R² = 1 - (∑_(i=1)^n (y_i - ŷ_i)²)/(∑_(i=1)^n (y_i - ȳ)²)
```

---

## Ф.7 — Прирост 1ПМ у начинающего (синтетическая модель)

**Раздел:** 4.2 «Набор данных» (синтетическая генерация)  
**Описание:** Еженедельный прирост 1ПМ в синтетических данных для спортсмена с опытом E (месяцев).

**LaTeX:**
```latex
\Delta_{1RM}^{week} = \frac{k}{1 + \alpha \cdot E}
```

где k — начальный прирост (кг/нед.), α — коэффициент замедления, E — стаж (мес.)

**UnicodeMath:**
```
Δ_(1RM)^week = k/(1+α·E)
```

---

## Ф.8 — Стратегия Last-Write-Wins (логическое условие)

**Раздел:** 5.3 «Протокол синхронизации»  
**Описание:** Условие принятия входящей записи при конфликте двух версий.

**LaTeX:**
```latex
\text{принять входящую} \iff UpdatedAt_{in} > UpdatedAt_{local}
```

**UnicodeMath:**
```
принять входящую ⟺ UpdatedAt_in > UpdatedAt_local
```

---

## Ф.9 — Предикат push-батча (фильтр записей)

**Раздел:** 5.3.3 «Цикл синхронизации»  
**Описание:** Условие включения локальной записи в push-батч при синхронизации.

**LaTeX:**
```latex
\text{pushBatch} = \{ r \in Local \mid r.UpdatedAt \geq LastSyncUtc \}
```

**UnicodeMath:**
```
pushBatch = {r ∈ Local | r.UpdatedAt ≥ LastSyncUtc}
```

## Примечание о вставке формул в Word

1. Все формулы вставляются через `Вставка → Уравнение` (Word 2016+).
2. В поле уравнения переключитесь в режим **«Линейный»** (кнопка на вкладке «Уравнение»).
3. Введите UnicodeMath-текст из этого файла.
4. Word автоматически отобразит профессиональную математическую нотацию.

Нумерация формул в тексте: (4.1), (4.2), ... — по разделу.
