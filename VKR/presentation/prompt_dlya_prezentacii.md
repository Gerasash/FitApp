КОНТЕКСТ ПРОЕКТА
================
- Название работы: «Разработка кроссплатформенного приложения для отслеживания силовых тренировок с локальной ML-моделью прогнозирования одноповторного максимума (1ПМ) и облачной синхронизацией».
- Рабочее название продукта: FitApp.
- ВУЗ: МТУСИ (Московский технический университет связи и информатики).
- Год защиты: 2026.
- Язык: русский.
- Стек: .NET MAUI 9 (клиент, Android + Windows), ASP.NET Core 9 (сервер), PostgreSQL/Neon, LightGBM → ONNX Runtime для ML.
- Ключевые особенности: offline-first, локальный ML без подписки, планировщик нагрузки, синхронизация Last-Write-Wins, open source.

ИНСТРУМЕНТ
==========
pptxgenjs (Node.js). Установка:
    npm install -g pptxgenjs
Запуск:
    node build_presentation_mtusi.js

Запретные приёмы pptxgenjs:
  • НЕ использовать «#» в hex-цветах (только "372579", не "#372579").
  • НЕ кодировать прозрачность в hex (использовать свойство opacity).
  • НЕ использовать unicode-буллеты «•» (только bullet: true или { code: "25A0" }).
  • НЕ переиспользовать объект options между addShape/addText вызовами (мутируется).
  • Использовать charSpacing (не letterSpacing) — оно молча игнорируется.
  • margin: 0 для текстовых блоков, выровненных с фигурами.

В конце — раздел «Возможные вопросы комиссии и ответы» (7 вопросов с развёрнутыми ответами на 3–5 предложений каждый).

ДОПОЛНИТЕЛЬНО
=============

## Как переиспользовать

1. Скопировать раздел «ПРОМТ» в новый чат.
2. Если изменились данные ВКР (другие цифры, новые слайды, иное название) — отредактировать соответствующие блоки до запуска.
3. Запросить «выполни этот промт».
4. После генерации — проверить визуально в PowerPoint (`presentation_zaschita_mtusi.pptx`), при необходимости подкрутить координаты в `.js`-файле и перезапустить `node build_presentation_mtusi.js`.

## Источник дизайн-правил

Палитра, шрифт и размеры взяты из официального шаблона МТУСИ (`template_mtusi.pptx`), слайды 2–4 которого содержат регламент:
- Слайд 2 — общие правила (4 пункта)
- Слайд 3 — типографика (шрифт Montserrat + размеры)
- Слайд 4 — палитра HEX
