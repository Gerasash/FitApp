## Шаблоны ссылок на таблицы

### Перед таблицей с данными / результатами

```
Результаты сравнения приведены в таблице X.Y.
Характеристики датасета представлены в таблице X.Y.
Сводные метрики качества модели приведены в таблице X.Y.
Параметры сравниваемых решений приведены в таблице X.Y.
Результаты тестирования сведены в таблицу X.Y.
```

### Ссылка в скобках (встроенная в предложение)

```
Модель LightGBM сокращает MAE на 39,1% относительно наивного предиктора (таблица X.Y).
Прирост 1ПМ соответствует эмпирическим нормам подготовки непрофессиональных спортсменов (таблица X.Y).
Все сценарии пройдены успешно (таблица X.Y).
```

### Ссылка через «в соответствии с»

```
В соответствии с таблицей X.Y, наилучший результат показывает модель LightGBM.
В соответствии с таблицей X.Y, все нефункциональные требования выполнены.
```

### Ссылка при интерпретации данных

```
Как видно из таблицы X.Y, …
Данные таблицы X.Y свидетельствуют о том, что …
Из таблицы X.Y следует, что …
Анализ данных таблицы X.Y показывает, что …
```

### Ссылка на таблицу в Приложении

```
Полные метрики в разрезе стажа приведены в таблице Г.1.
Детальные результаты представлены в таблице Г.1 (Приложение Г).
В соответствии с таблицей Б.1, …
```
